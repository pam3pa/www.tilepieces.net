How to add and remove css rules

Tilepieces allows you to easily change the text and images of your html documents.

It also allows you to modify the structure using different interfaces (called panels), including the html inspector interface similar to that present in the browser developer tool.

Through the components, you can easily add functionality to your html documents.
By importing this example of the well-known CKEditor library, it will be possible to make changes using a WYSIWYG that you are perhaps more used to editing!
